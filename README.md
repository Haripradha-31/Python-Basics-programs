# 🐍 Python Basics Programs

A beginner-friendly collection of Python programs that cover the fundamental concepts of Python programming. This repository is designed for students, beginners, and anyone starting their Python learning journey.

## 📚 About

This repository contains simple Python programs that demonstrate basic programming concepts such as:

- Printing Output
- Variables
- Data Types
- User Input
- Arithmetic Operations
- Basic Calculator

These programs are written in a simple and easy-to-understand manner, making them suitable for beginners.

---

## 📂 Programs Included

| S.No | Program Name | Description |
|------|--------------|-------------|
| 1 | Hello World | Prints "Hello, World!" |
| 2 | Print Name | Displays the user's name |
| 3 | Variables | Introduction to variables |
| 4 | Data Types | Demonstrates Python data types |
| 5 | User Input | Takes input from the user |
| 6 | Addition | Adds two numbers |
| 7 | Subtraction | Subtracts two numbers |
| 8 | Multiplication | Multiplies two numbers |
| 9 | Division | Divides two numbers |
| 10 | Simple Calculator | Performs Addition, Subtraction, Multiplication, and Division |

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/Python-Basics-Programs.git
```

### Navigate to the Project Folder

```bash
cd Python-Basics-Programs
```

### Run Any Program

```bash
python filename.py
```

Example:

```bash
python 01_HelloWorld.py
```
